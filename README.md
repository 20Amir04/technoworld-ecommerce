
# TechnoWorld E-commerce

## 📌 About the Project
TechnoWorld is a full-stack responsive e-commerce web application for selling professional and gaming computers, laptops, and smartphones.

The project was built as a portfolio full-stack application to practice building a real-world online store using modern technologies including React, TypeScript, TailwindCSS, ASP.NET Core Web API, and Entity Framework Core.

The application includes a full shopping flow: browsing products, searching, adding items to cart, managing wishlist, user authentication, and order placement.

## 🚀 Features

### 🛒 E-commerce functionality

- Product catalog with categories and subcategories
- Product detail page
- Add to Cart
- Cart management
- Checkout flow
- Order placement
- Order history in user account

### 👤 User system

- User registration
- User login
- Edit personal information
- Delete account
- Authentication with protected routes

### ❤️ Wishlist

- Add / remove products from wishlist
- Persistent wishlist state

### 🔍 Product Search

- Keyword search across products
- Real-time filtering of results

### 🔔 UX Enhancements

- Toast notifications when adding items to cart
- Mobile responsive navigation (burger menu)
- Responsive layout for all pages

### 📱 Responsive Design

- Desktop layout
- Tablet layout
- Mobile layout

## ⚙️ Installation

### Clone repository
  git clone https://github.com/20amir04/technoworld-ecommerce.git |
  cd technoworld-ecommerce

### Run Backend
  cd backend |
  dotnet restore |
  dotnet run |

### Run Frontend
  cd frontend |
  npm install |
  npm run dev |
